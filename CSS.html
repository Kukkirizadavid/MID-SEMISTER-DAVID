CSS

CSS describes how HTML elements are to be displayed on screen, paper, or in other media. OR CSS saves a lot of work. It can control the layout of multiple web pages all at once

The following are ways how CSS can be inserted into a web page

  • Inline Styles With inline styles, style sheet information is applied directly to the HTML element. Instead of defining the style once, then applying the style against all instances of an element .
  • Embedded Styles You add all CSS information to one part of the document (usually the top). This allows you to style any element on the page from a single place. You do this by embedding the CSS information within tags in the head of your document.
  • External styles sheets are the most common method of applying styles to awebsite. Most modern websites an external stylesheet to apply site wide styles to the whole website
  • Import Styles You can also use the CSS @import rule to import an external style sheet. To do this, use the style tag.

CSS SELECTORS

The following are CSS selectors

  1. Simple selectors (select elements based on name, id, class)
  2. Combinator selectors (select elements based on a specific relationship between them)
  3. Pseudo-class selectors (select elements based on a certain state)
  4. Pseudo-elements selectors (select and style a part of an element)
  5. Attribute selectors (select elements based on an attribute or attribute value)
